# Cyrillic Languages

A multilingual alphabet reference by Paratype — a knowledge base of about 80 Cyrillic-based languages, maintained as a set of hand-edited JSON files and a Python pipeline that compiles them into the data served by the public site.

**Live site:** <https://paratype.github.io/cyrillic-languages/>
_This repository is a working copy; the canonical published version lives under the `paratype` organization._

## About the project

The project has existed since the 2000s and is continuously updated to reflect changes in the languages' alphabets and their Unicode support. For each language, the knowledge base captures not only the current alphabet but also additional characters needed to support the language — historical forms, dialect glyphs, and lexicographic notations — useful for historical books, textbooks, and reference material.

As a reference, it is intended for font designers, typographers, and anyone working with Cyrillic.

The aggregated pan-Cyrillic character set that supports every language in the project is generated alongside the per-language data.

## Scope

This documentation focuses on **the Cyrillic script library**. A Latin-script tree exists in the repository (`cyrillic-languages/library/latin/`) but is an internal work-in-progress and is not currently open for external contributions.

## Repository map

The repository mixes two different things: the **data** (open to contributors) and the **site engine / generated artifacts** (operator-only). Contributors should only ever edit files in the data area.

**Open for contributions**

- `cyrillic-languages/library/cyrillic/base/*.json` — per-language source files. This is where the work happens.
- `CONTRIBUTING.md` / `CONTRIBUTING-RU.md` — this documentation itself.
- `docs/` — maintainer-facing notes; typo fixes and clarifications welcome.

**Operator-only — do not edit in a contributor PR**

- `cyrillic-languages/site/**` — generated by the pipeline from the sources above.
- `cyrillic-languages/static/**`, `cyrillic-languages/fonts/**`, `cyrillic-languages/index.html`, `cyrillic-languages/asset-manifest.json`, `cyrillic-languages/favicon.ico`, `cyrillic-languages/robots.txt`, root `index.html` — the web-site engine bundle.
- `cyrillic-languages/scripts/**` — the Python pipeline.
- `cyrillic-languages/languages.json`, `cyrillic-languages/locales.json`, `cyrillic-languages/glyphs_list_categories.json`, `cyrillic-languages/library/cyrillic/cyrillic_library.json`, `cyrillic-languages/library/cyrillic/sortorder_cyrillic.txt`, `cyrillic-languages/unicode14.txt`, `cyrillic-languages/PT_PUA_unicodes*.txt` — shared configuration and reference tables. If you think one of these needs to change for your contribution, open an issue describing the need instead of a PR.
- `cyrillic-languages/library/cyrillic/_legacy/**` — archived historical language files; see the README inside that folder.

## Contributing

If you want to correct or extend a language entry — fix a missing character, update a description, refine an alphabet — the workflow is:

1. Fork this repository.
2. Edit only files under `cyrillic-languages/library/cyrillic/base/`.
3. Open a Pull Request against `master`.

Full instructions, the JSON schema reference, the glyph-marker catalog, and a pre-flight checklist live in **[CONTRIBUTING.md](CONTRIBUTING.md)** (English) and **[CONTRIBUTING-RU.md](CONTRIBUTING-RU.md)** (Russian).

Maintainer / operator notes — how the pipeline is run, how to add a new language, how to maintain the sort order and Unicode description tables — are in **[docs/MAINTAINING.md](docs/MAINTAINING.md)**.
